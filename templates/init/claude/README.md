# .claude

Этот каталог предназначен для project-local assets под Claude.

Рекомендуемое содержимое:

- `agents/` — project-specific agents
- `commands/` — slash-команды проекта
- `skills/` — project-local skills, если проект решит их хранить в репозитории

Это versioned слой проекта. Его содержимое можно коммитить и ревьюить.
