# .codex

Этот каталог используется в этом проекте как repo-level convention для
Codex-specific assets.

Важно:

- основной repo-level instruction contract для Codex по-прежнему задается через
  `AGENTS.md`
- `.codex/` здесь рассматривается как project-local место для дополнительных
  материалов, которые могут понадобиться Codex workflow

Рекомендуемое содержимое:

- `agents/`
- `prompts/`
- `routing/`
- `skills/`

Это именно проектная конвенция `ai-teamlead`, а не внешний стандарт Codex.
